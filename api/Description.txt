Bu Python script'i, Dynatrace API'lerinden host, process, process group ve servis verilerini çekmek için 
tasarlanmıştır. 
Script aşağıdaki adımları gerçekleştirir:

1. API çağrıları: Dynatrace altyapısı hakkında bilgi içeren verileri almak için çeşitli Dynatrace API uç 
noktalarına (hostlar, process'ler, process grupları ve servisler) HTTP GET istekleri gönderir.

2. Veri dönüşümü: Alınan verileri, json formatına dönüştürür.

3. Veri seçimi: İstenilen sütunları json verilerinden seçer ve "[ ]" ve "{ }" gibi işaretleri gerektiği 
şekilde temizler.

4. Veri ekleme: Seçilen verileri boş bir listeye ekler.

5. DataFrame oluşturma: Listedeki verileri dataframe yapısına dönüştürür.

6. DataFrame düzenleme: Process, process group ve servis DataFrame'lerine diğer DataFrame'lerde bulunan bazı 
sütunları eklemek için merge fonksiyonlarını kullanır. Bu şekilde, DataFrame'lerde istenilen veri eklemeleri 
yapılır.

7. Excel oluşturma: Oluşturulan DataFrame'ler Excel dosyalarına (.xlsx) dönüştürülür ve işletim sisteminde 
belirtilen dosya yoluna yazdırılır.
